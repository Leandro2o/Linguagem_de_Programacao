# Linguagem de Programação

> Repositório destinado ao estudo, exercícios, projetos e conceitos práticos sobre Linguagem de Programação.

Este espaço armazena implementações de algoritmos, estudos sobre sintaxe, paradigmas de programação (orientação a objetos, estruturada, funcional) e soluções desenvolvidas durante o aprendizado da disciplina/módulo de **Linguagem de Programação**.

---

## Conteúdo do Repositório

O repositório está organizado de forma a facilitar o acesso aos tópicos estudados:

* **`Exercicios/`**: Exercícios de fixação de sintaxe e lógica.
* **`Projetos/`**: Pequenas aplicações e desafios práticos desenvolvidos.

---

## Linguagens e Tecnologias
